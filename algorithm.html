Injectable therapy decision algorithm

This page explains the decision logic used by the CDS prototype for adults with type 2 diabetes (T2D). The goal is transparency: the tool generates a recommendation plus rationale and next steps.

Algorithm version: v1 (MVP) Scope: first injectable, intensification, simplification Engine: py/engine.py

Textual description of the algorithm

1) Country selection (first node)

The algorithm first applies a country profile (RU, TR, LB, JO, IQ, EU, US, OTHER). This profile may affect availability assumptions and country-specific notes.

  • Jordan (JO): FRC (iGlarLixi) is treated as not available.
  • Turkey (TR): if FRC is recommended and BMI < 35, show a comment: “reimbursement often requires BMI ≥35; otherwise may be out-of-pocket” (comment only).

2) Severe hyperglycaemia

If catabolic symptoms are present OR HbA1c ≥ 10%, recommend insulin start or urgent insulin intensification.

Rationale: rapid glycaemic control is prioritised in severe hyperglycaemia/catabolic context to reduce acute risk.

3) Simplification / de-intensification (complex insulin regimens)

If the patient is on basal-bolus or premix insulin and has recurrent hypoglycaemia or high regimen complexity/adherence issues:

  • If FRC is available (and not Jordan) → switch to FRC (iGlarLixi).
  • Otherwise → simplify within insulin options.

Rationale: de-intensification can be appropriate when hypoglycaemia risk or treatment burden outweighs benefit.

4) Intensification on basal insulin

If the patient is on basal insulin and HbA1c remains above target:

  1. If GLP-1 RA is available → add GLP-1 RA to basal insulin.
  2. Else if FRC is available (and not Jordan) → switch basal insulin to FRC (iGlarLixi).
  3. Else → add prandial insulin (basal-plus strategy, stepwise escalation).

Rationale: GLP-1 RA is typically preferred before adding mealtime insulin when feasible; FRC provides basal+postprandial control with one injection.

5) First injectable selection (no current injectable therapy)

A. Cardiorenal disease (ASCVD / HF / CKD):

  • If GLP-1 RA available → start GLP-1 RA.
  • Otherwise → start basal insulin.

B. Obesity-driven approach (BMI ≥ 30):

  • If GLP-1 RA available → start GLP-1 RA.

C. Higher HbA1c without catabolic symptoms:

  • If HbA1c > 9 and FRC available (not Jordan) → start FRC (iGlarLixi).

D. Default rule:

  • If GLP-1 RA available → start GLP-1 RA.
  • Otherwise → start basal insulin.
