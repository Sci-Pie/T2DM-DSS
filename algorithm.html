Iraq — Textual description

Shared gate — Severe hyperglycaemia

If catabolic symptoms are present or HbA1c ≥ 10%, the algorithm immediately recommends start / intensify insulin. This overrides all other branches.

Step 0 — First injectable selection

The HbA1c gap (current HbA1c − individualised target) and BMI determine the starting injectable:

  • Gap < 2%, BMI ≤ 30: Basal insulin & titration. 2nd-generation basal insulins (degludec, glargine U-300) preferred.
  • Gap < 2%, BMI > 30: GLP-1 RA alone (preferred) or BI + GLP-1 RA (FRC or separately).
  • Gap ≥ 2%, BMI ≤ 30: BI + GLP-1 RA (FRC or separately) — or premix agents ♯ if other options are not accessible locally.
  • Gap ≥ 2%, BMI > 30: BI + GLP-1 RA (FRC or separately).

Step 1 — Escalation from basal-only (gap < 2%, BMI ≤ 30 ladder)

If HbA1c remains above target on basal insulin alone: escalate to BI + GLP-1 RA (FRC preferably, or separately).

Step 2 — Add rapid-acting insulin

If HbA1c remains above target on any of the following: GLP-1 RA alone · BI + GLP-1 RA (FRC or separately):

→ BI (max dose) + GLP-1 RA + Rapid-acting insulin.

Titrate basal insulin to its maximum tolerated dose; add prandial rapid-acting insulin starting at the largest meal.

Step 3 — Final intensification

If HbA1c remains above target on BI (max) + GLP-1 RA + rapid-acting insulin:

  • Basal-bolus: optimise basal + rapid-acting insulin before each main meal.
  • Premixed insulin: twice-daily premixed as a simpler alternative.

Standing notes (always shown for Iraq)

  • GLP-1 RA choice should be guided by patient-specific considerations: established CVD, CKD, desired weight benefit, access and cost (affordability).
  • Basal insulin: 2nd-generation agents (degludec, glargine U-300) are preferred over older generations due to lower hypoglycaemia risk.
  • ♯ Premix agents: complex insulin regimens (such as premix insulins) may be used as alternatives if other options are not accessible locally.

Turkey — Textual description

1) Severe hyperglycaemia override

Catabolic symptoms or HbA1c ≥ 10% → start / intensify insulin immediately.

2) Intensification on FRC

On FRC, target unmet → add rapid-acting insulin (may be off-label).

On FRC + rapid, target unmet → basal-bolus or premixed insulin.

3) Simplification

On basal-bolus or premix with recurrent hypoglycaemia → switch to FRC.

4) Intensification on basal insulin

On basal insulin, target unmet or PPG uncontrolled → switch to FRC.

5) First injectable

  • Gap < 2%, BMI ≤ 30 → basal insulin.
  • Gap < 2%, BMI > 30 → FRC.
  • Gap ≥ 2% (any BMI) → FRC.

Turkey reimbursement note: if FRC is recommended and BMI < 35 kg/m², treatment may be out-of-pocket.

Shared logic (all countries)

Severe hyperglycaemia gate

Applied before any country-specific logic. If catabolic symptoms are present or HbA1c ≥ 10%, the engine immediately returns "Start / intensify insulin (severe hyperglycaemia)".

Default HbA1c target

If the user does not provide an individualised HbA1c target but does provide a current HbA1c value, the engine uses a default target of 7.0% and adds a comment to the output.

Gap calculation

gap = HbA1c (current) − HbA1c (effective target). Used as the primary routing variable for first injectable selection.
